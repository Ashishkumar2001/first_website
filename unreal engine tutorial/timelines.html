Timelines


Timelines are used to create simple animations, such as changing the location, rotation, or color of an object. To add a new Timeline Node, right-click in the graph, select “Add Timeline...” from the bottom of the menu, and give the Timeline a name. To edit a Timeline, double-click it to open it in the Timeline Editor.



Timeline Options

To the right of the create track buttons, there are a few options you can set for your Timeline. The first box allows you to set the length of the Timeline. To the right of that is the Use Last Keyframe? option. If this is unchecked, the very last tick of the animation will be ignored. This is useful when looping, to prevent skipping in the animation when the loop starts over.
