Static Meshes


A mesh is a 3D model of an object. There are two specific types of meshes that you can use as Actors in the Unreal Engine. These are the Static Mesh and the Skeletal Mesh.

There are a few different Static Mesh Actors in Place Mode of the Modes Panel. Under the Basic tab, there are Static Meshes in the shape of a cube, sphere, cylinder, and cone. These can be dragged into the Level, and then positioned, rotated, and scaled as desired.
